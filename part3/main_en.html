What is Content-Type in the HTTP request and why do we need it?

The Content-Type header is used to indicate the media type of the resource enclosed in the message body. It tells the server what type of data is being sent in the body of the request.

For example:

  • text/plain: Plain text data.
  • text/html: HTML document.
  • application/json: JSON data.
  • image/jpeg: JPEG image.

Its used for:

  • Server Processing.
  • Interpretation by Recipient.
  • Security Measures.
  • Content Negotiation.
